# CyphLab Task Manager — Full-Stack Platform

A modular, decoupled project and team task management suite engineered with a robust role-based access control (RBAC) architecture. This application utilizes a clean RESTful interface separating a responsive client dashboard application from a secure, stateless API service engine.
